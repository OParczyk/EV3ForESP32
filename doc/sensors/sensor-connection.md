
# Notes on Sensor connection

## Pinout

### EV3 Sensor

NC -> 2
GND -> 3,1
VCC5V -> 4
TX -> 6
RX -> 5

### NXT Sensor

NC (9V, 5V Pullup) -> 1
GND -> 3,2
VCC5V -> 4
SDA -> 6
SCL -> 5

### EV3 Motor

Tacho1 (5V) -> 5
Tacho2 (5V) -> 6
GND -> 3
VCC5V -> 4
Motor1 -> 1
Motor2 -> 2

### Cabel

White (M1) -> 1
Black (M2) -> 2
Red (GND) -> 3
Green (5V) -> 4
Yellow (DATA1) -> 5
Blue (DATA2) -> 6
